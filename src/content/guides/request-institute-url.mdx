# Request for Institute Subdomain (.nitk.ac.in)


> *Note: Only Faculty can make this request (can request on behalf of students).*


## How to Apply

1. Go to [IRIS](https://iris.nitk.ac.in/)
2. Navigate to **Service Requests**
3. Select **Request for Institute URL**

## Request Flow

Faculty → HoD → CCC Chairman
