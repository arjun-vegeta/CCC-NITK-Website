# Request for Captive Portal ID

## How to Apply

1. Go to [IRIS](https://iris.nitk.ac.in/)
2. Navigate to **Service Requests**
3. Select **Request for Captive Portal ID**

## Request Flow

Faculty → HoD → CCC Chairman

> *Note: Only Faculty can make this request.*
