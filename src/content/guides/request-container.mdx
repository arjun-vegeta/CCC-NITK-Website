# Request for Container/ Virtual Machine

<NoteBox type="warning">
> *Note: Only Faculty can request this (can request on behalf of students).*
</NoteBox>

## How to Apply

1. Go to [IRIS](https://iris.nitk.ac.in/)
2. Navigate to **Institute → Service Requests**
3. Select **Request for Container**

## Request Flow

Faculty → HoD → CCC Chairman
